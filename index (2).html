One-time setup — takes about 5 minutes. You only do this once, then share the link with Adam.

  1. Go to script.google.com and click New project
  2. Delete any existing code, then paste in the contents of Code.gs (the other file I gave you)
  3. Click Save (the floppy disk icon)
  4. Click Deploy → New deployment
  5. Click the gear icon next to "Type" and choose Web app
  6. Set Execute as: Me — set Who has access: Anyone — click Deploy
  7. Sign in with Google when prompted, click Allow
  8. Copy the Web app URL and paste it below

This URL is saved on your device. Share it with Adam so he can bookmark the same page.
